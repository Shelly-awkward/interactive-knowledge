書く （かく） to write
一類動詞（五段） ／ ない形：書かない （ない前は「か」＝あ段 → 一類）
書く kaku to write
書かない かかない negative
書いた kaita wrote
書かなかった kakanakatta did not write
書きます kakimasu write (polite)
書きません kakimasen not write (polite)
書いて kaite write, and...
書いてください kaite kudasai please write
書いている kaite iru is writing
書こう kakou let's write
書きましょう kakimashou let's write (polite)
書け kake write! (command)
書くな kakuna don't write
書けば kakeba if you write
書ける kakeru can write
書かれる kakareru be written
書かせる kakaseru make/let write
書かされる kakasareru forced to write
